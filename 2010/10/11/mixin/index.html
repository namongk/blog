Mixin

In object-oriented programming languages, a mixin is a class that provides a certain functionality to be inherited by a subclass, while not meant for instantiation (the generation of objects of that class).

이 정의에 따르면, 추상 클래스(abstract class) 여러 개를 다중 상속할 수 있으면 그게 바로 mixin이다.
